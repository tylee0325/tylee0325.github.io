개인정보 처리방침

시행일: 2026년 3월 7일

팽구 - English Writing Coach(이하 "확장 프로그램")는 개인정보 보호를 중요하게 생각합니다. 본 방침은 확장 프로그램의 개인정보 처리에 관한 사항을 안내합니다.

1. 수집하는 개인정보

이 확장 프로그램은 사용자의 개인정보를 직접 수집하지 않습니다.

설정 정보(AI 모델 선택, 테마 등)는 사용자의 브라우저 로컬 저장소(chrome.storage.local)에만 저장되며, 외부 서버로 전송되지 않습니다.

2. API 키 저장

사용자가 입력하는 AI API 키(OpenAI, Google AI)는 브라우저 로컬 저장소에만 저장됩니다. 개발자는 이 키에 접근할 수 없습니다.

API 키는 교정 및 연습 기능 호출 시에만 사용되며, 해당 AI 서비스 제공자에게 직접 전송됩니다.

3. 텍스트 데이터 처리

교정을 요청하는 텍스트는 사용자가 설정한 AI 서비스(OpenAI 또는 Google Gemini)에 전송됩니다. 이 데이터는:

  • 사용자의 자체 API 키를 통해 전송됩니다
  • 개발자의 서버를 경유하지 않습니다
  • 각 AI 서비스 제공자의 개인정보 처리방침에 따라 처리됩니다

관련 방침:

  • OpenAI 개인정보 처리방침
  • Google 개인정보 처리방침

4. 교정 히스토리

교정 결과는 사용자의 브라우저 로컬 저장소에 최대 100건까지 저장됩니다. 이 데이터는 외부로 전송되지 않으며, 사용자가 직접 삭제할 수 있습니다.

5. 제3자 제공

AI 서비스 호출을 제외하고, 사용자 데이터를 외부에 제공하지 않습니다.

6. 아동 개인정보

이 확장 프로그램은 아동의 개인정보를 수집하지 않습니다.

7. 방침 변경

본 방침이 변경될 경우 이 페이지를 통해 공지합니다.
